# MCP Hello World Server

A simple Model Context Protocol (MCP) server implementation written in TypeScript.

## Features

- MCP protocol support
- Custom tools (e.g., greetings)
- TypeScript based
- Stdio transport

## Requirements

- Node.js (v18 or higher)
- npm

## Installation

```bash
git clone <repo-url>
cd mcp-hello-world-server
npm install
```

## Build

```bash
npm run build
```

## Run

```bash
npm start
```
